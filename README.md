💻 Playwright Laptop E-Commerce Automation
A robust automation testing framework for a Laptop e-commerce platform, focusing on scalability, maintainability, and AI-driven efficiency.

🛠 Tech Stack
Language: TypeScript

Framework: Playwright

Pattern: Page Object Model (POM)

CI/CD: GitHub Actions

Reporting: Playwright HTML Reporter

✨ Key Features
POM Architecture: Decoupled UI locators from business logic for easy maintenance and scalability.

AI-Enhanced Testing: Leveraging AI tools for smart locator generation and handling dynamic web elements.

End-to-End (E2E) Suites: Automated critical user flows: Search -> Add to Cart -> Checkout.

Smart Waiting: Implemented Playwright’s auto-waiting and custom assertions to eliminate flaky tests.

CI/CD Integration: Automated test execution on every GitHub push.
